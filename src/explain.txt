src/config.py
Loads settings from the .env file.
It keeps all configuration in one clean object, such as dataset path, LLM provider, and max iterations.

src/dataset.py
Loads the Bitext CSV and normalizes the important columns.
It also contains the real data logic: list categories, list intents, count rows, get examples, and prepare rows for summaries.

src/schemas.py
Defines the Pydantic input schemas for the tools.
This tells the LLM exactly what arguments each tool accepts.

src/tools.py
Defines the tools the agent can use to query the dataset.
Each tool has a clear name, description, input schema, and calls methods from BitextDataset.

src/router.py
Classifies the user query before the agent starts working.
It decides whether the query is structured, unstructured, or out_of_scope.

src/graph.py
Builds the LangGraph flow.
It connects the router, agent node, tool node, and fallback node into one working agent loop.

src/cli.py
Implements the command-line chat interface.
It lets you ask questions interactively and prints the reasoning trace, including tool calls and tool outputs.
